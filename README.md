# Restaurant Website Demo

This is a clean, responsive restaurant website built using HTML and CSS.

It is designed to help restaurants showcase their menu, attract customers, and allow easy contact through WhatsApp.

## Features
- Responsive design (mobile, tablet, desktop)
- Menu section
- About section
- Contact section
- WhatsApp contact button

## Live Preview
 https://ohlie-dev.github.io/restaurant-website-demo/

## Purpose
This project is a demo to show how a restaurant business can have a simple, professional online presence.
